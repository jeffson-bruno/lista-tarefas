# Lista de Tarefas (To-Do List) em PHP

Aplicação simples de **Lista de Tarefas** feita com:

- PHP (procedural, sem framework)
- Bootstrap
- jQuery
- JavaScript

## Funcionalidades (planejadas)

- Login simples (admin)
- Adicionar tarefas
- Listar tarefas
- Marcar como concluída
- Editar tarefa
- Excluir tarefa

Projeto criado para portfólio e prática de PHP + Front-end.
